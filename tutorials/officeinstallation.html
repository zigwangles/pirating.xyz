Install Office

Once you’ve downloaded the Office image file (img), follow these steps to install it:

  1. Mount the img file by double-clicking it in Windows 8/10 or right-clicking and selecting Mount.
  2. In the mounted folder, navigate to the Office folder and run Setup64.exe for 64-bit systems or Setup.exe for 32-bit systems.

Office Activation Methods

There are two recommended methods for activating Office:

MAS Activation Process

MAS uses a remote KMS host for activation, leaving no binaries on your system, reducing antivirus detection risks. However, the remote KMS host may go offline.

  1. Download MAS from its GitHub Repo and extract the files.
  2. Run the All-In-One-Version.cmd file as an administrator.
  3. If your Windows is activated with KMS38, protect it from being overwritten by regular KMS activation by using the Extras option.
  4. Select the Online KMS Activation option to activate Office for 180 days. Enable the Renewal Task to automatically renew the activation periodically.

KMS_VL_ALL Activation Process

KMS_VL_ALL sets up a local KMS host on your machine, so it doesn’t rely on an external server for activation. However, local binaries may trigger antivirus warnings.

  1. Download the KMS_VL_ALL_AIO.cmd file from the Codepen page or MyDigitalLife forums.
  2. Run the cmd file as an administrator.
  3. If your Windows is activated with KMS38, disable the option Process Windows to avoid affecting the Windows activation.
  4. Choose the option to Install Activation Auto-Renewal for continuous activation.

Final Notes

Both MAS and KMS_VL_ALL convert your Office installation from a retail to a volume license, enabling KMS activation. If you use Office 365, these methods will activate a Mondo license, allowing you to use local features.
